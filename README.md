# Projeto 01 — Sistema Inteligente de RH com IA

Sistema modular de RH que usa **automação (Make)** e **IA com prompts estruturados** para apoiar triagem, recrutamento, suporte ao RH e análise de clima organizacional.

Dados 100% fictícios/simulados.

## Conteúdo do repositório
- `docs/` — documentação do projeto
- `prompts/` — prompts estruturados por módulo
- `schemas/` — modelos JSON (entrada/saída)
- `examples/` — exemplos fictícios
- `automations/` — exports do Make/n8n
- `assets/` — diagramas e prints

## Módulos do sistema
1. Triagem de Currículos (MVP)
2. Automação de Recrutamento
3. Assistente de RH com IA
4. Análise de Clima Organizacional
